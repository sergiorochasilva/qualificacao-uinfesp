During the development of this work, several preliminary experiments were conducted before the final architecture was selected. This appendix summarizes the main configurations that were tried and the behavior observed in each case, so the progression toward the final model remains explicit.

\section{Experiment 1}

The first attempt used a multilayer neural network to analyze the audio input in the time domain and estimate the corresponding FM synthesis parameters for resynthesis. The main characteristics were:

\begin{enumerate}
\item 1000 audio input samples, each 3 seconds long.
\item Each audio input was split into frames of 2048 samples, resulting in 43 frames per audio sample.
\item The network was divided into two parts: a feature extractor, implemented with a time-distributed layer and three dense layers, and a regressor, implemented with three dense layers to estimate the FM synthesis parameters from the extracted features.
\item The results were poor, and the model suffered from overfitting despite several tested variations.
\end{enumerate}

\section{Experiment 2}

The second attempt replaced the time-distributed baseline with a 1D Convolutional Neural Network to extract features directly from the waveform and then estimate the FM synthesis parameters with a multilayer regressor. The main characteristics were:

\begin{enumerate}
\item 1000 audio input samples, each 1 second long, to reduce memory usage.
\item The audio inputs were not split into frames.
\item The network was divided into two parts: a feature extractor with three convolutional layers, and a regressor with three dense layers to estimate the FM synthesis parameters from the extracted audio features.
\item The results improved over Experiment 1, but they were still poor. The model also suffered from even stronger overfitting despite several tested variations.
\end{enumerate}

\section{Experiment 3}

Experiment 3 was similar to Experiment 2, but with one fewer convolutional layer and MaxPooling layers added to the feature extractor. The main characteristics were:

\begin{enumerate}
\item 1000 audio input samples, each 1 second long.
\item The network was divided into two parts: a feature extractor composed of convolutional and pooling layers, and a regressor composed of three dense layers used to estimate the FM synthesis parameters from the extracted audio features.
\item The results were similar to the previous experiments.
\end{enumerate}

\section{Experiment 4}

This experiment was similar to Experiments 2 and 3, but it used two branches of convolutional layers in the feature-extraction stage. Pooling layers were also used, with the goal of capturing distinct features and recognizing different frequency patterns in the audio input. The main characteristics were:

\begin{enumerate}
\item 1000 audio input samples, each 1 second long.
\item The network was divided into two main parts: a feature extractor and a regressor. The feature extractor itself was composed of two parallel convolutional branches.
\item The results were slightly worse than in the previous experiment, but still similar. This time, however, the model did not suffer from overfitting.
\end{enumerate}
